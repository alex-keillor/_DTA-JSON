DTA Developer examination for Alexander Keillor.

This is an atttempt at a wireframe or a proof of concept for an app that can consume a JSON feed for a product catalogue

Note: all logic is client based. All data access logic and processing is client side.

    Hence the reader is asked to be mindful of the following:
  • no vaidation due to time constraints
  • no server side components or dependencies are provided BUT would be essential to any production implmentation
  • there is no database or business logic layers - all logic is in the client. Again this makes this a partial solution in the authors view
  • this is not a real shopping cart - very important to remember ;-)
  • there is no payment gateway integration into merchange baking systems, paypal, bitcoin interchange, etc. again requires a robust and secure production environment
  • etc, etc and ad nauseam to re-affirm this is a wireframe or mockup

Click the Menu option Products to access an AJAX representation of an online shopping site based on the supplied JSON products feed.

Built using a Visual Studio Code on a Windows 10 Machine in HTML5, Bootstrap JS Framework and custom AJaX/AHaH
* AJAX = Asynchronous JavaScript and XML
* AHAH = Asynchronous HTML and HTTP
{This solution is based on similar code patterns using HTML snippets and JavaScript that the author applied to win a university award as a rudimentary single page application}

This was lots of fun to play with and learn/revise key concepts ... thanks as this excercise has been fun!
